# Node API Tasks
API node desenvolvida com propósitos de estudo e aprendizagem. 

## Modo de uso
Antes de executar o projeto devem ser rodados os seguintes comandos no terminal 

~~~
npm i express
npm i dotenv
npm i mongoose
~~~
